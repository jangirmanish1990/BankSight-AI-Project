# Spec 01 — Data Setup & Database Initialisation

**Project:** BankSight AI — Agentic Banking Analytics Assistant
**File:** `.claude/specs/01-data-setup.md`
**Purpose:** Step-by-step instructions for loading, cleaning, and
initialising `data/banking_mock.db` from raw source files.
**Read by:** Claude Code before running `data/setup_db.py`

---

## Overview

The database `data/banking_mock.db` is the **single source of truth**
for all agents. It contains 4 tables loaded from raw CSV files with
specific cleaning transformations applied during setup.

```
data/
├── raw/
│   ├── customers.csv         ← Kaggle: Bank Customer Churn dataset
│   ├── transactions.csv      ← Kaggle: Credit Card Spending India
│   ├── loan_emi.csv          ← Kaggle: Bank Loan Status dataset
│   └── employee_perf.csv     ← Generated via Faker (Python)
└── banking_mock.db           ← Final SQLite database (output)
```

## Step 1 — Load & Clean `customers` Table

**Source:** `data/raw/customers.csv` (3,225 rows, 16 columns)

### Cleaning Transformations (apply in this exact order)

**1a. Replace European locations with Indian cities**
```python
indian_cities = [
    'Delhi', 'Mumbai', 'Bengaluru', 'Gurugram',
    'Pune', 'Hyderabad', 'Chennai', 'Kolkata'
]
df['city'] = df['city'].apply(lambda x: random.choice(indian_cities))
df['country'] = 'India'
```

**1b. Convert USD amounts to INR (multiply by 83)**
```python
df['balance']           = (df['balance'] * 83).round(2)
df['estimated_salary']  = (df['estimated_salary'] * 83).round(2)
```

**1c. Add missing columns**
```python
df['account_type'] = random.choices(
    ['Savings', 'Current', 'Salary'],
    weights=[60, 25, 15], k=len(df)
)
df['kyc_status'] = random.choices(
    ['Verified', 'Pending'],
    weights=[85, 15], k=len(df)
)
```

**1d. Standardise column names**
```python
# Rename to match BankSight schema
df.rename(columns={
    'products_number': 'products_number',
    'credit_card':     'credit_card',
    'active_member':   'active_member'
}, inplace=True)
```

**1e. Validate before loading**
```python
assert df['churn'].isin([0, 1]).all(),        "Invalid churn values"
assert df['nps_score'].between(0, 10).all(),  "NPS out of range"
assert df['credit_score'].between(300, 900).all(), "Credit score OOR"
assert df.isnull().sum().sum() == 0,          "Nulls found"
```

## Step 2 — Load & Clean `transactions` Table

**Source:** `data/raw/transactions.csv`
**Target:** 50,000 rows covering Jan 2024 – Dec 2024

### Cleaning Transformations

**2a. Add customer_id foreign key**
Link transactions to customers by randomly assigning
`customer_id` values from the customers table:
```python
customer_ids = customers_df['customer_id'].tolist()
df['customer_id'] = random.choices(customer_ids, k=len(df))
```

**2b. Standardise category names**
```python
category_map = {
    'Food':          'Food',
    'Travel':        'Travel',
    'Shopping':      'Shopping',
    'Bills':         'Utilities',
    'Entertainment': 'Entertainment',
    'Fuel':          'Travel',
    'Grocery':       'Food'
}
df['category'] = df['category'].map(category_map).fillna('Shopping')
```

**2c. Add missing columns**
```python
df['payment_mode'] = random.choices(
    ['Credit Card', 'Debit Card', 'UPI', 'NetBanking'],
    weights=[40, 20, 30, 10], k=len(df)
)
df['status'] = random.choices(
    ['Success', 'Failed', 'Reversed'],
    weights=[92, 5, 3], k=len(df)
)
```

**2d. Add `is_anomaly` flag**
Flag transactions where amount > 2× that customer's
monthly average spend:
```python
monthly_avg = df.groupby(
    ['customer_id', df['transaction_date'].str[:7]]
)['amount'].transform('mean')
df['is_anomaly'] = (df['amount'] > 2 * monthly_avg).astype(int)
```

**2e. Convert amounts to INR**
```python
# If source amounts are in USD
df['amount'] = (df['amount'] * 83).round(2)
# Clip to realistic INR range: ₹500 to ₹1,50,000
df['amount'] = df['amount'].clip(500, 150000)
```

### Expected Output
```
Table: transactions
Rows:     ~50,000
Columns:  9
Date range: Jan 2024 – Dec 2024
Anomalies: ~5–8% flagged (is_anomaly = 1)
```

## Step 3 — Load & Clean `loan_emi` Table

**Source:** `data/raw/loan_emi.csv`
**Target:** 400 rows (approximately 40% of customers have loans)

### Cleaning Transformations

**3a. Assign customer_id foreign key**
```python
# Pick 400 unique customer_ids from customers table
loan_customers = random.sample(customer_ids, 400)
df['customer_id'] = loan_customers
```

**3b. Standardise loan type values**
```python
loan_type_map = {
    'MORTGAGE':  'Home',
    'PERSONAL':  'Personal',
    'AUTO':      'Auto',
    'CREDIT':    'Credit Card',
    'HOME':      'Home'
}
df['loan_type'] = df['loan_type'].str.upper().map(loan_type_map)
```

**3c. Standardise EMI status values**
```python
status_map = {
    'Fully Paid':   'Paid',
    'Charged Off':  'Missed',
    'Current':      'Paid',
    'Late':         'Delayed',
    'Default':      'Missed'
}
df['emi_status'] = df['loan_status'].map(status_map)
```

**3d. Convert amounts to INR**
```python
df['loan_amount'] = (df['loan_amnt'] * 83).round(2)
df['emi_amount']  = (df['loan_amount'] / df['term_months']).round(2)
```

**3e. Add missing date columns**
```python
df['loan_start_date'] = pd.to_datetime(
    df['issue_d'], format='%b-%Y'
).dt.strftime('%Y-%m-%d')
df['loan_end_date'] = (
    pd.to_datetime(df['loan_start_date']) +
    pd.DateOffset(months=df['term_months'])
).dt.strftime('%Y-%m-%d')
df['emi_due_date'] = '15'  # EMI due on 15th of each month
```

## Step 4 — Generate `employee_performance` Table

**Source:** Auto-generated using Faker — no raw CSV needed
**Target:** 50 rows × 12 months = 600 rows total

```python
from faker import Faker
fake = Faker('en_IN')   # Indian locale for realistic names

departments = ['Loans', 'Cards', 'Support', 'Risk', 'Operations']
regions     = ['North', 'South', 'East', 'West']
grades      = ['A', 'B', 'C', 'D']
months      = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun',
               'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec']

rows = []
for emp_id in range(1, 51):
    for month in months:
        target   = random.randint(1000000, 5000000)
        achieved = int(target * random.uniform(0.6, 1.3))
        grade    = (
            'A' if achieved >= target * 1.1 else
            'B' if achieved >= target else
            'C' if achieved >= target * 0.8 else 'D'
        )
        rows.append({
            'employee_id':          f'E{emp_id:03d}',
            'name':                 fake.name(),
            'department':           random.choice(departments),
            'region':               random.choice(regions),
            'target_amount':        target,
            'achieved_amount':      achieved,
            'quality_score':        random.randint(60, 100),
            'customer_complaints':  random.randint(0, 20),
            'month':                month,
            'year':                 2024,
            'performance_grade':    grade
        })

df = pd.DataFrame(rows)
```

### Expected Output
```
Table: employee_performance
Rows:     600 (50 employees × 12 months)
Columns:  11
Grade split: ~25% A / 35% B / 25% C / 15% D
```

## Step 5 — Load All Tables to SQLite

After all 4 DataFrames are cleaned and validated,
load them into `banking_mock.db` in this exact order
(customers first — other tables depend on its customer_id):

```python
from sqlalchemy import create_engine

engine = create_engine('sqlite:///data/banking_mock.db')

customers_df.to_sql(
    'customers', engine, if_exists='replace', index=False)
transactions_df.to_sql(
    'transactions', engine, if_exists='replace', index=False)
loan_emi_df.to_sql(
    'loan_emi', engine, if_exists='replace', index=False)
employee_df.to_sql(
    'employee_performance', engine, if_exists='replace', index=False)

print("✅ banking_mock.db initialised successfully")
```

## Step 6 — Post-Setup Validation

Run these checks after loading to confirm database integrity:

```python
import sqlite3
conn = sqlite3.connect('data/banking_mock.db')

checks = {
    'customers row count':        "SELECT COUNT(*) FROM customers",
    'churned customers':          "SELECT COUNT(*) FROM customers WHERE churn=1",
    'transactions row count':     "SELECT COUNT(*) FROM transactions",
    'anomaly flagged txns':       "SELECT COUNT(*) FROM transactions WHERE is_anomaly=1",
    'loan_emi row count':         "SELECT COUNT(*) FROM loan_emi",
    'missed EMI count':           "SELECT COUNT(*) FROM loan_emi WHERE emi_status='Missed'",
    'employee rows':              "SELECT COUNT(*) FROM employee_performance",
    'orphan transactions':        """SELECT COUNT(*) FROM transactions t
                                    LEFT JOIN customers c
                                    ON t.customer_id = c.customer_id
                                    WHERE c.customer_id IS NULL"""
}

for label, query in checks.items():
    result = conn.execute(query).fetchone()[0]
    print(f"{label}: {result}")

conn.close()
```

## Important Rules

1. **Always run setup_db.py before starting any agent session**
2. **Never run INSERT/UPDATE/DELETE directly on banking_mock.db** —
   all data changes must go through setup_db.py
3. **If database gets corrupted**, delete banking_mock.db and
   re-run setup_db.py from scratch
4. **Seed random operations** with `random.seed(42)` for
   reproducible mock data across team members
5. **customers table must load first** — all other tables
   have foreign key dependency on customer_id
